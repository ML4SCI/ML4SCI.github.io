## About ML4SCI

Machine Learning for Science (ML4SCI) is an open-source organization that brings together modern machine learning techniques and applies them to cutting-edge problems in Science, Technology, Engineering, and Mathematics (STEM).

## ML4SCI in GSoC 2025

The ML4SCI open-source organization is participating in the **2025 Google Summer of Code**.

ML4SCI is an umbrella organization that welcomes other projects and organizations related to machine learning for science.
